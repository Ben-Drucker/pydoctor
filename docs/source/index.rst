Introduction
============

Welcome to ``pydoctor``'s documentation!


Installation
------------

``pydoctor`` can be installed from PyPI::

   $ pip install -U pydoctor

Install pydoctor with ``restructuredtext`` support::

   $ pip install -U pydoctor[rst]
